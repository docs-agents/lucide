# Lucide

Lucide 是一个开源图标库，提供 1000+ 矢量（SVG）文件，用于在数字和非数字项目中显示图标和符号。该库旨在帮助设计师和开发人员更轻松地将图标集成到他们的项目中，提供了多个官方[软件包](https://lucide.dev/packages)，使在项目中使用这些图标变得更加便捷。

## 关于品牌标识

Lucide **不接受**品牌标识，我们也不打算在未来添加它们。这是由于**法律限制**、**设计一致性考虑**和**实际维护原因**的综合因素。

[点击此处阅读我们关于 Lucide 中品牌标识的官方声明。](./BRAND_LOGOS_STATEMENT.md)
